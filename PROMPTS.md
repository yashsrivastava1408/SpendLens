# Anthropic API Prompt

The only AI feature in SpendLens is the generation of a personalized 100-word summary paragraph based on the hardcoded audit logic results.

## Final Prompt
**Model:** `claude-3-haiku-20240307`

```text
System: You are a concise financial analyst specializing in AI infrastructure spend.
Write exactly 100 words. No markdown. Plain text only.

User: Summarize this AI spend audit in 100 words:
Team size: {teamSize} | Use case: {useCase}
Tools audited: {toolsList}
Total current monthly spend: ${currentSpend}
Total potential monthly savings: ${monthlySavings}
Top recommendation: {topRecommendation}

Write a personalized summary that: names the specific tools, states the savings opportunity clearly, gives one concrete action item, and ends with a forward-looking sentence.
Be direct and specific. Avoid generic advice.
```

## Rationale & Iteration History

1. **Attempt 1:**
   - *Prompt:* "Write a summary of this user's AI spend based on these numbers: {data}. Tell them what to do."
   - *Result:* Failed. It hallucinated pricing data that contradicted our deterministic audit engine, and wrote a massive 400-word essay with bullet points.

2. **Attempt 2:**
   - *Prompt:* Added "Do not hallucinate prices. Keep it under 100 words."
   - *Result:* Better, but it kept formatting the output as a Markdown list, which broke the UI aesthetic on the frontend. It also lacked a specific action item.

3. **Final Attempt (Current):**
   - *Fixes:* Explicitly enforced "No markdown. Plain text only." and "Write exactly 100 words." by moving it to the System Prompt, which models follow much more strictly. By feeding it the `topRecommendation` string (which is generated by our deterministic engine), it stopped hallucinating and strictly summarized the hard math.

## Fallback Mechanism
If the Anthropic API fails (network error, rate limit, invalid key), the application gracefully degrades to a template string. The template interpolates the exact same variables, ensuring a seamless user experience. A broken AI feature is worse than no AI feature.
